## 开始使用

### 1.安装

```sh
yarn add vueel-demo
```

### 2.使用

#### 安装依赖 Project setup

```
yarn install
```

#### 启动服务 Compiles and hot-reloads for development

```
yarn serve
```

#### 生产构建 Compiles and minifies for production

```
yarn build
```

#### 运行测试 Run your unit tests

```
yarn test:unit
```

#### 格式化代码 Lints and fixes files

```
yarn lint
```

See [Configuration Reference](https://cli.vuejs.org/config/).
